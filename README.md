# 📊 AI-Automated Analyst Dashboard

## Overview
The AI-Automated Analyst Dashboard is an interactive, AI-powered data analysis tool designed to bridge the gap between raw data and actionable insights. This application allows users to upload datasets and instantly generate visualizations, perform segmentation, and interact with their data using natural language — all without writing a single line of code.

Built using a modern data stack, the dashboard integrates data processing, machine learning, and large language models (LLMs) into a seamless user experience. The goal is to democratize advanced analytics and make data-driven decision-making accessible to users across domains.

---

## 🚀 Key Features
- Upload and analyze **CSV and Excel datasets**
- Automatic **data ingestion and statistical summaries**
- **K-Means clustering** for unsupervised segmentation
- Interactive **Plotly visualizations** with filtering
- AI-powered **natural language insights**
- Context-aware responses using **LLM integration**
- Clean and responsive UI built with **Streamlit**

---

## 🏗️ Architecture Overview
The system follows a modular pipeline architecture:

1. **Data Input Layer**  
   Users upload structured datasets (CSV/XLSX)

2. **Data Processing Layer**  
   Data ingestion, schema detection, and statistical summaries

3. **Analytics & ML Engine**  
   K-Means clustering and cluster profiling

4. **AI Insights Layer**  
   LLM generates context-aware insights from summaries + queries

5. **Presentation Layer**  
   Streamlit dashboard with visualizations and chat interface

---

## 🛠️ Tech Stack
- **Frontend/UI:** Streamlit  
- **Data Processing:** Pandas  
- **Machine Learning:** Scikit-learn (K-Means)  
- **Visualization:** Plotly Express  
- **AI/LLM:** Hugging Face Inference API (Qwen 2.5-7B-Instruct)  
- **Deployment:** Streamlit Community Cloud
